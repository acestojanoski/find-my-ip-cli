# find-my-ip-cli

CLI to find your IP address

It is using the find-my-ip-api.

Github link: https://github.com/acestojanoski/find-my-ip-api

API specs: https://find-my-ip.herokuapp.com/api-specs

## Installation

`npm install -g find-my-ip-cli`

## Usage

`find-my-ip`
